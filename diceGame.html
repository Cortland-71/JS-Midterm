TIPS

Let's go over what we just did.


If I said something like (a + a) you would understand this to equate to the number 6. Variables can change to something else of course but in Java it's important to remember that the kind of data that can be stored in a variable must be mentioned explicitly.


Let's look at an example...

Integers

In Java you must define the type of variable you are making. In this example let's look at one of the most commonly used types, the int. int stands for Integer and notice that it is lower case. int can hold positive or negative whole number between a specific amount. When creating a variable you must tell the compiler what type of variable you are making. You can also assign a value to the variable in the same line as shown below.


int 
a
 = 
4
;


In this example the "int a" portion is our variable declaration. It's telling java that we want to make a variable called "a" and the type is an integer. The "= 4" is our value assignment. The "=" means assign and the "4" is the value.
